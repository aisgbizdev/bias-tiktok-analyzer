# BIAS Pro Privacy Policy

**Last updated:** November 2025  
**Applies to:** BIAS Pro GPTs & BIAS TikTok Analyzer API  

---

### Overview
BIAS Pro is a Behavioral Intelligence Analysis system that analyzes **public creator behavior** such as voice, tone, and social media presence.  
We do **not collect, store, or share any personal or sensitive data**.

---

### What Data We Process
- Only **publicly available TikTok profile data** (followers, likes, bio, etc.)  
- No passwords, cookies, or authentication data are used.  
- Data is processed **temporarily** only during the analysis request.

---

### Data Storage & Retention
All requests are processed in real time and not stored or shared.  
We do not keep or log any user-generated queries or TikTok profile data.

---

### External APIs
The system uses this API endpoint:  
`https://bias-tiktok-analyzer.onrender.com`  
which only retrieves **public TikTok information** and never saves any personal data.
